> アプデしたらダークモード消えた……😭
【iPhoneで、Xのダークモードを使いたい！】

アプリ「ショートカット」を開く
→下部の「オートメーション」を選択
→右上の「＋」をタップ
→真ん中へんにある「アプリ」を選択
→“いつ”の画面で、アプリの選択からXを選ぶ
→「開いている」にチェック(※)
→「すぐに実行」にチェックして「次へ」をタップ
→“Xが開かれたとき”の画面で、「開始」の下にある「新規ショートカットを作成」をタップ
→「アクションを検索」の欄に「外観」と打てば「外観モードを設定」が出てくるので、それをタップ
→外観モードを「ダーク」に「変更」を選んで右上のチェックボタンをタップ

これでできます！🤗

同じように、ショートカットで「Xを閉じたとき、外観モードをライトに変更」を作っておけば(※のところで「閉じている」を選ぶと、閉じるときの動作を決められます)、Xを開いてるときだけダークモード」環境のできあがり(一瞬のタイムラグは許してあげて)

> 🇨🇳 译文：更新后，深色模式消失了……😭
【我想在我的iPhone上使用X的深色模式！ ]

打开应用程序“快捷方式”
→ 选择底部的“自动化”
→点击右上角的“+”
→ 选择中间的“应用程序”
→ 在“何时”屏幕上，从应用程序选择中选择 X
→ 选中“打开”(*)
→ 勾选“立即运行”并点击“下一步”
→ 在“当 X 打开时”屏幕上，点击“开始”下的“创建新快捷方式”
→ 如果您在“搜索操作”字段中输入“外观”，则会出现“设置外观模式”，因此请点击它。
→选择“更改”为“深色”外观模式，然后点击右上角的对勾按钮

现在你可以了！ 🤗

同样，如果您创建一个快捷方式“当 X 关闭时，将外观模式更改为浅色”（在 * 处选择“关闭”，您可以决定关闭它时会发生什么），并且仅当 X 打开时您才会有深色模式（允许短暂的滞后）。
